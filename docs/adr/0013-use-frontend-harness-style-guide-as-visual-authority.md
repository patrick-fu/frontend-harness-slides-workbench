# Use the Frontend Harness Style Guide as visual authority

Use the latest Frontend Harness Slides Style index and per-Style Design DNA as
the sole visual authority for Workbench Styles. The Workbench owns runtime
Style IDs, bilingual names, and Bands, but does not copy the guide or infer
Style rules from existing Topics.

Coordinators may inspect existing Topics to plan portfolio diversity. Author
Workers read the official guide for their assigned Style but remain isolated
from other Workbench Topic implementations, preserving independent model
expression without creating a second Style source of truth.
