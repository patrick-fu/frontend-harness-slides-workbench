# Stadium Wave SVG Emoji assets

The two SVG files in this directory are local, deterministic copies of
OpenMoji 16.0 color assets:

- `openmoji-person-standing.svg` — Unicode `U+1F9CD`
- `openmoji-person-raising-hand.svg` — Unicode `U+1F64B`

All emojis designed by OpenMoji, the open-source emoji and icon project.
License: [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/).
Source: [OpenMoji](https://openmoji.org/).

The SVGs were optimized for whitespace and numeric precision; their visual form
was otherwise preserved. The slide changes pose through CSS transforms and
composes the standing figure with an original seat outline. These local copies
remain under CC BY-SA 4.0.
