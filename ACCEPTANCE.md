## 4. 风格合规性检查

每个风格必须通过以下检查：

### 4.1 接口契约

- [ ] 导出 `default` 组件（接受 `BespokeStyleProps`）
- [ ] 导出命名函数 `getMetadata(lang)` 返回 `StyleMetadata`
- [ ] `getMetadata("en")` 和 `getMetadata("zh")` 均返回完整结构
- [ ] 5 个 scene，每个有 1-N 个动态 beat

### 4.2 CSS 规范

- [ ] 所有尺寸使用 `cqw`/`cqh`（Stage 内禁用 `px`/`vw`/`vh`）
- [ ] CSS 变量使用 `--style-{XX}-*` 前缀
- [ ] 使用 CSS Modules (`.module.css`)
- [ ] `reducedMotion` 模式下动画时长为 0
- [ ] `isThumbnail` 模式下隐藏导航元素

### 4.3 导航规范

- [ ] 5 个 scene 导航指示器（非缩略图模式）
- [ ] 点击调用 `onNavigate(scene, 0)`
- [ ] `onClick` 中调用 `e.stopPropagation()`
- [ ] `aria-label` 标注

### 4.4 内容规范

- [ ] 双语内容（EN + ZH）
- [ ] Scene transition track 使用 `key={\`{id}-{scene}\`}` 强制 remount
- [ ] `isThumbnail=true` 时 `onNavigate` 可能为 `undefined`，不得崩溃

### 4.5 字体加载

- [ ] `useEffect` 注入 Google Fonts `<link>`
- [ ] 去重检查（`document.getElementById("style-{XX}-fonts")`）
- [ ] CJK 字体在 metadata 中使用 `cjk:` 前缀
