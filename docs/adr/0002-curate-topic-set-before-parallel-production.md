# Curate a Topic Set before production

A Topic Set is curated as a coherent cross-domain anthology before Topics are authored. Curation establishes Content Territory, Narrative Archetype, Viewing Mode, Visual Engine, Motion Language, Intensity, Navigation Language, Transition Score, and Signature Effect so independent authors produce complementary work rather than convergent variations.

## Considered Options

- Free-form Topic assignment.
- A single shared narrative.

Both weaken collection-level variety or Style fit.

## Consequences

Topic briefs are evaluated against the shared curation language before visual authoring begins.
